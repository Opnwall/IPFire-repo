Opnwall IPFire Community Repository

Community plugins for IPFire 2.29 x86_64

安装

curl -fsSL https://opnwall.github.io/IPFire-repo/install-opnwall.sh | sh
opnwall list
opnwall install adguardhome
opnwall remove adguardhome
opnwall upgrade
